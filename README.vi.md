# Agentic Coding Lab

Repo này chứa các tài liệu nghiên cứu và ghi chú về các giải pháp agentic coding.

## Tài liệu hiện có

- **understand-beads.md**: Tài liệu tìm hiểu về Beads.
- **understand-bmad/**: Thư mục chứa tài liệu tìm hiểu về BMAD method.
- **understand-claude-code-newbie.md**: Tài liệu tìm hiểu về Claude Code (cho người mới).

## Mục tiêu

Lưu trữ và tổng hợp kiến thức về các phương pháp, công cụ và quy trình phát triển phần mềm với sự hỗ trợ của AI Agent.
